# String ends with?

## DESCRIPCIÓN
Completa la solución para que devuelva `true` si el primer argumento (cadena) termina con el segundo argumento (también cadena).

### Ejemplos:
```c
solution('abc', 'bc') // returns true
solution('abc', 'd')  // returns false
```
