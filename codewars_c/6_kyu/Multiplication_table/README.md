# Multiplication_table

## DESCRIPCIÓN
Su tarea es crear una tabla de multiplicar N×N, del tamaño proporcionado en el parámetro.

### Ejemplos:
Por ejemplo, cuando se da `size`3:

```
1 2 3
2 4 6
3 6 9
```


```
[[1,2,3],[2,4,6],[3,6,9]]
```

Nota: en C, debe devolver una tabla asignada de filas asignadas
